# Plot a caretList object

This function plots the performance of each model in a caretList object.

## Usage

``` r
# S3 method for class 'caretList'
plot(x, metric = NULL, ...)
```

## Arguments

- x:

  a caretList object

- metric:

  which metric to plot

- ...:

  ignored

## Value

A ggplot2 object
